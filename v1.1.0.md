### 4. صفحات جديدة أخرى
- **نسيت كلمة المرور** — `/forgot-password` مع نموذج إدخال البريد الإلكتروني

### 5. تحسين واجهة تسجيل الدخول
- واجهة محدثة مع تصميم متدرج (gradient bg)
- أيقونات داخل حقول الإدخال (Mail, Lock)
- زر إظهار/إخفاء كلمة المرور
- خيار "تذكرني"
- أزرار تسجيل دخول عبر Google و Facebook
- رسائل خطأ متحركة
- حالة التحميل (loading spinner)

### 6. تحسين واجهة التسجيل
- نموذج من خطوتين: اختيار نوع الحساب ← إدخال البيانات
- شريط تقدم يوضح الخطوات
- بطاقات اختيار نوع الحساب (خريج / صاحب عمل) مع أيقونات وتأثيرات hover
- جميع حقول الإدخال مع أيقونات
- حقل تأكيد كلمة المرور
- خانة الموافقة على الشروط والأحكام
- أزرار رجوع وإرسال

### 7. تحسين نظام CSS
- دمج جميع مكونات CSS في `globals.css` باستخدام `@layer components`
- كلاسات موحدة: `btn-primary`, `btn-secondary`, `btn-ghost`, `btn-primary-lg`, `btn-secondary-lg`, `glass-card`, `stat-card`, `input-field`, `badge-*`
- تصميم متجاوب كامل لجميع الشاشات
- دعم الوضع المظلم (dark mode) عبر Tailwind

### 8. إنشاء مستخدمين اختباريين
| اسم المستخدم | كلمة المرور | الدور |
|---|---|---|
| `admin` | `admin` | مدير النظام |
| `company` | `Company123` | صاحب عمل |
| `user` | `User1234` | خريج |
